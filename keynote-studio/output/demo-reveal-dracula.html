Keynote Studio

Reveal.js + Dracula 示例

单文件 HTML 演示稿（可继续模板化）

目标

  • 用 Web 前端技术生成演示稿
  • 交付单个 HTML 文件
  • 后续支持自动化批量生成

结构示例

  1. 封面页
  2. 目录页
  3. 内容页
  4. 总结页

代码块示例

const deck = {
  engine: 'reveal.js',
  theme: 'dracula',
  output: 'single-html'
};

console.log('Keynote Studio demo ready');
