# ai-global-news

全球 AI 资讯聚合系统（采集、清洗、去重、分类、摘要、检索、订阅推送）。

## 当前状态

- 已完成：项目初始化、需求定义、MVP 验收标准、来源优先级、执行计划
- 下一步：后端骨架（FastAPI + SQLite3 + 迁移）

## 目录

- `backend/`：FastAPI + 任务调度 + 数据处理
- `frontend/`：Next.js 前端
- `infra/`：部署与环境配置
- `docs/`：设计文档、里程碑、上线清单
- `data/`：本地样例数据与中间产物
- `scripts/`：运维与开发脚本
