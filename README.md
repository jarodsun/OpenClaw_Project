# 项目根目录 - Project Root

这是所有个人项目的集中管理目录。

## 📁 目录结构规范

```
~/.openclaw/project/
├── README.md          # 本文件 - 目录使用说明
├── project-a/         # 项目 A 的独立文件夹
├── project-b/         # 项目 B 的独立文件夹
└── ...                # 更多项目...
```

## ✅ 项目添加规范

1. **为每个新项目创建独立文件夹**
   ```bash
   mkdir ~/path/to/project-name
   ```

2. **在本文件中添加项目说明**
   - 在 `## 项目列表` 部分添加项目条目
   - 包含项目名称、简要描述、目录路径

3. **项目内部自定义结构**
   - 各项目可在自己的文件夹内自由组织结构
   - 推荐包含 `README.md`、`src/`、`docs/` 等标准目录
