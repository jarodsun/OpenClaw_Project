# IdleWoool

Web 放置挂机 RPG 项目。

## 技术栈
- 前端：Next.js
- 后端：FastAPI
- 数据库：SQLite3

## 目录结构
- `docs/`：项目文档（需求、系统、架构、测试、发布）
- `frontend/`：Next.js 前端项目
- `backend/`：FastAPI 后端项目
- `backend/alembic/`：数据库迁移
- `backend/tests/`：后端测试
- `data/`：数值配置与静态数据
- `assets/`：美术与音频资源
- `scripts/`：研发与运维脚本
